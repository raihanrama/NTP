ABOUT US

Standar Waktu Indonesia

Waktu Standar Indonesia adalah waktu resmi untuk Indonesia yang diberikan oleh Coordinated Universal Time (UTC). Ditambah tujuh/delapan/sembilan jam berdasarkan zona waktu Indonesia.

UTC +7/+8/+9 | Longitude 120° East

Standar Nasional Satuan Ukur Waktu dan Frekuensi

Standar Nasional Satuan Ukur Waktu dan Frekuensi telah ditunjuk sebagai pencatat waktu nasional oleh Kabinet Indonesia dan sebagai Laboratorium Standar Pengukuran Nasional dalam Sistem Pengukuran Nasional.

HOW IT WORKS

--:--:--

Jam web ini disinkronisasikan dengan server NTP resmi BSN (ntp.bsn.go.id)

Server UTC(IDN) melakukan sinkronisasi dengan Jam Atom Cesium SNSU hingga akurasi milidetik

Jam Atom Cesium SNSU secara terus-menerus dibandingkan dengan UTC melalui metode tampilan umum GPS untuk penelusuran skala waktu internasional

Jam Atom Cesium yang berada di Laboratorium SNSU sangat menjaga tingkat akurasi waktu hingga ketepatan waktu satu detik dalam 60.000 tahun

Panduan Sinkronisasi NTP

Sinkronisasi di Windows

  1. Buka Command Prompt sebagai Administrator
  2. Ketik perintah berikut: w32tm /config /manualpeerlist:ntp.bsn.go.id /syncfromflags:manual /reliable:yes /update
  3. Restart layanan waktu dengan perintah: net stop w32time && net start w32time
  4. Sinkronkan waktu dengan perintah: w32tm /resync /force

Sinkronisasi di Linux

  1. Install NTP jika belum terinstall: sudo apt-get install ntp
  2. Edit file konfigurasi NTP: sudo nano /etc/ntp.conf
  3. Tambahkan server BSN: server ntp.bsn.go.id prefer
  4. Restart layanan NTP: sudo service ntp restart

Sinkronisasi di MacOS

  1. Buka Terminal
  2. Ketik perintah berikut: sudo sntp -sS ntp.bsn.go.id
  3. Masukkan password administrator Anda
  4. Tunggu hingga proses sinkronisasi selesai
